# Kubernetes 实践指南

本书将介绍 Kubernetes 相关实战经验与总结，助你成为一名云原生老司机。

本书为电子书形式，内容为本人多年的云原生与 Kubernetes 实战经验进行系统性整理的结果，不废话，纯干货。
